–í–æ—Ç –ø—Ä–æ—Å—Ç–æ–µ –æ–±—ä—è—Å–Ω–µ–Ω–∏–µ –º–µ—Ç–æ–¥–æ–≤ –ø—Ä–µ–¥–æ–±—Ä–∞–±–æ—Ç–∫–∏ –¥–∞–Ω–Ω—ã—Ö —Å –Ω–∞–≥–ª—è–¥–Ω—ã–º–∏ –ø—Ä–∏–º–µ—Ä–∞–º–∏ –Ω–∞ Python:

### 2. **–û–±—Ä–∞–±–æ—Ç–∫–∞ –≤—ã–±—Ä–æ—Å–æ–≤**
**–ü—Ä–æ–±–ª–µ–º–∞:** –ù–µ–∫–æ—Ç–æ—Ä—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è —è–≤–Ω–æ "—Å–ª–æ–º–∞–Ω—ã".

- **–ü–æ—Ä–æ–≥–æ–≤—ã–π –º–µ—Ç–æ–¥:**  

- **IQR-–º–µ—Ç–æ–¥:** –ê–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–æ–µ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –≤—ã–±—Ä–æ—Å–æ–≤

**–ü—Ä–∏–º–µ—Ä –≤—ã–±—Ä–æ—Å–∞:**  
–£ 99% –∏–≥—Ä–æ–∫–æ–≤ PS –æ—Ç 80 –¥–æ 150, –Ω–æ –æ–¥–∏–Ω –∏–º–µ–µ—Ç PS 999 (–æ—á–µ–≤–∏–¥–Ω–æ –æ—à–∏–±–∫–∞).

**–ö–≤–∞–Ω—Ç–∏ÃÅ–ª—å** –≤ –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–æ–π —Å—Ç–∞—Ç–∏—Å—Ç–∏–∫–µ ‚Äî –∑–Ω–∞—á–µ–Ω–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ –∑–∞–¥–∞–Ω–Ω–∞—è —Å–ª—É—á–∞–π–Ω–∞—è –≤–µ–ª–∏—á–∏–Ω–∞ –Ω–µ –ø—Ä–µ–≤—ã—à–∞–µ—Ç —Å —Ñ–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω–æ–π –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å—é. –ï—Å–ª–∏ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –∑–∞–¥–∞–Ω–∞ –≤ –ø—Ä–æ—Ü–µ–Ω—Ç–∞—Ö, —Ç–æ –∫–≤–∞–Ω—Ç–∏–ª—å –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è –ø—Ä–æ—Ü–µ–Ω—Ç–∏–ª–µ–º –∏–ª–∏ –ø–µ—Ä—Ü–µ–Ω—Ç–∏–ª–µ–º. –ù–∞–ø—Ä–∏–º–µ—Ä, —Ñ—Ä–∞–∑–∞ ¬´90-–π –ø—Ä–æ—Ü–µ–Ω—Ç–∏–ª—å –º–∞—Å—Å—ã —Ç–µ–ª–∞ —É –Ω–æ–≤–æ—Ä–æ–∂–¥–µ–Ω–Ω—ã—Ö –º–∞–ª—å—á–∏–∫–æ–≤ —Å–æ—Å—Ç–∞–≤–ª—è–µ—Ç 4 –∫–≥¬ª –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ 90 % –º–∞–ª—å—á–∏–∫–æ–≤ —Ä–æ–∂–¥–∞—é—Ç—Å—è —Å –≤–µ—Å–æ–º, –º–µ–Ω—å—à–∏–º –ª–∏–±–æ —Ä–∞–≤–Ω—ã–º 4 –∫–≥, –∞ 10 % –º–∞–ª—å—á–∏–∫–æ–≤ —Ä–æ–∂–¥–∞—é—Ç—Å—è —Å –≤–µ—Å–æ–º, –±–æ–ª—å—à–∏–º –ª–∏–±–æ —Ä–∞–≤–Ω—ã–º 4 –∫–≥

### **IQR-–º–µ—Ç–æ–¥: –û–±—ä—è—Å–Ω–µ–Ω–∏–µ –ø—Ä–æ—Å—Ç—ã–º–∏ —Å–ª–æ–≤–∞–º–∏ —Å –ø—Ä–∏–º–µ—Ä–∞–º–∏**  

**IQR (Interquartile Range)** ‚Äî —ç—Ç–æ —Å—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫–∏–π –º–µ—Ç–æ–¥ –¥–ª—è –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–æ–≥–æ –ø–æ–∏—Å–∫–∞ –≤—ã–±—Ä–æ—Å–æ–≤ –≤ –¥–∞–Ω–Ω—ã—Ö.  
–í—ã–±—Ä–æ—Å—ã ‚Äî —ç—Ç–æ –∑–Ω–∞—á–µ–Ω–∏—è, –∫–æ—Ç–æ—Ä—ã–µ —Å–∏–ª—å–Ω–æ –æ—Ç–ª–∏—á–∞—é—Ç—Å—è –æ—Ç –æ—Å–Ω–æ–≤–Ω–æ–π –º–∞—Å—Å—ã –¥–∞–Ω–Ω—ã—Ö (–Ω–∞–ø—Ä–∏–º–µ—Ä, –æ—à–∏–±–∫–∏ –≤–≤–æ–¥–∞ –∏–ª–∏ –∞–Ω–æ–º–∞–ª–∏–∏).  

---

#### **–ö–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç IQR-–º–µ—Ç–æ–¥?**  
1. **–†–∞–∑–±–∏–≤–∞–µ–º –¥–∞–Ω–Ω—ã–µ –Ω–∞ –∫–≤–∞—Ä—Ç–∏–ª–∏:**  
   - **Q1 (25-–π –ø–µ—Ä—Ü–µ–Ω—Ç–∏–ª—å)** ‚Äî –∑–Ω–∞—á–µ–Ω–∏–µ, –Ω–∏–∂–µ –∫–æ—Ç–æ—Ä–æ–≥–æ –Ω–∞—Ö–æ–¥—è—Ç—Å—è 25% –¥–∞–Ω–Ω—ã—Ö.  
   - **Q3 (75-–π –ø–µ—Ä—Ü–µ–Ω—Ç–∏–ª—å)** ‚Äî –∑–Ω–∞—á–µ–Ω–∏–µ, –Ω–∏–∂–µ –∫–æ—Ç–æ—Ä–æ–≥–æ –Ω–∞—Ö–æ–¥—è—Ç—Å—è 75% –¥–∞–Ω–Ω—ã—Ö.  
   - **IQR = Q3 - Q1** ‚Äî —ç—Ç–æ "—Ä–∞–∑–º–∞—Ö" —Å—Ä–µ–¥–Ω–∏—Ö 50% –¥–∞–Ω–Ω—ã—Ö.  

2. **–û–ø—Ä–µ–¥–µ–ª—è–µ–º –≥—Ä–∞–Ω–∏—Ü—ã –Ω–æ—Ä–º–∞–ª—å–Ω—ã—Ö –∑–Ω–∞—á–µ–Ω–∏–π:**  
   - **–ù–∏–∂–Ω—è—è –≥—Ä–∞–Ω–∏—Ü–∞** = `Q1 - 1.5 * IQR`  
   - **–í–µ—Ä—Ö–Ω—è—è –≥—Ä–∞–Ω–∏—Ü–∞** = `Q3 + 1.5 * IQR`  

3. **–í—Å—ë, —á—Ç–æ –≤—ã—Ö–æ–¥–∏—Ç –∑–∞ —ç—Ç–∏ –≥—Ä–∞–Ω–∏—Ü—ã ‚Äî –≤—ã–±—Ä–æ—Å—ã.**  

---

#### **–ü—Ä–∏–º–µ—Ä –∏–∑ –∫–æ–¥–∞**  
–î–æ–ø—É—Å—Ç–∏–º, —É –Ω–∞—Å –µ—Å—Ç—å —Å—Ç–æ–ª–±–µ—Ü `PS` (–Ω–∞–ø—Ä–∏–º–µ—Ä, —Å–∏–ª–∞ –∏–≥—Ä–æ–∫–æ–≤ –≤ –∏–≥—Ä–µ):  

```python
Q1 = df['PS'].quantile(0.25)  # 25% –¥–∞–Ω–Ω—ã—Ö ‚â§ Q1 (–Ω–∞–ø—Ä–∏–º–µ—Ä, Q1 = 80)
Q3 = df['PS'].quantile(0.75)  # 75% –¥–∞–Ω–Ω—ã—Ö ‚â§ Q3 (–Ω–∞–ø—Ä–∏–º–µ—Ä, Q3 = 150)
IQR = Q3 - Q1                 # IQR = 150 - 80 = 70

# –ì—Ä–∞–Ω–∏—Ü—ã –≤—ã–±—Ä–æ—Å–æ–≤:
lower_bound = Q1 - 1.5 * IQR  # 80 - 1.5*70 = -25 (–Ω–æ —Å–∏–ª–∞ –Ω–µ –º–æ–∂–µ—Ç –±—ã—Ç—å <0, –ø–æ—ç—Ç–æ–º—É —Ä–µ–∞–ª—å–Ω–∞—è –Ω–∏–∂–Ω—è—è –≥—Ä–∞–Ω–∏—Ü–∞ 0)
upper_bound = Q3 + 1.5 * IQR  # 150 + 1.5*70 = 255
```

**–§–∏–ª—å—Ç—Ä–∞—Ü–∏—è –≤—ã–±—Ä–æ—Å–æ–≤:**  
```python

# –û—Å—Ç–∞–≤–ª—è–µ–º —Ç–æ–ª—å–∫–æ –∏–≥—Ä–æ–∫–æ–≤ —Å PS –æ—Ç 0 –¥–æ 255
```

---

#### **–ü—Ä–∏–º–µ—Ä –≤—ã–±—Ä–æ—Å–∞**  
–î–æ–ø—É—Å—Ç–∏–º, —É –Ω–∞—Å —Ç–∞–∫–∏–µ –¥–∞–Ω–Ω—ã–µ:  
- –£ **99% –∏–≥—Ä–æ–∫–æ–≤** —Å–∏–ª–∞ (`PS`) **–æ—Ç 80 –¥–æ 150**.  
- –ù–æ **–æ–¥–∏–Ω –∏–≥—Ä–æ–∫** –∏–º–µ–µ—Ç `PS = 999` (–æ—á–µ–≤–∏–¥–Ω–∞—è –æ—à–∏–±–∫–∞ –∏–ª–∏ —á–∏—Ç–µ—Ä).  

**–ü—Ä–∏–º–µ–Ω—è–µ–º IQR:**  
- `Q1 = 80`, `Q3 = 150` ‚Üí `IQR = 70`  
- `–í–µ—Ä—Ö–Ω—è—è –≥—Ä–∞–Ω–∏—Ü–∞ = 150 + 1.5*70 = 255`  

**–†–µ–∑—É–ª—å—Ç–∞—Ç:**  
- `PS = 999` **–≤—ã—à–µ 255** ‚Üí —ç—Ç–æ –≤—ã–±—Ä–æ—Å, –µ–≥–æ –Ω—É–∂–Ω–æ —É–¥–∞–ª–∏—Ç—å.

#### **–í—ã–≤–æ–¥**  
IQR-–º–µ—Ç–æ–¥ –ø–æ–º–æ–≥–∞–µ—Ç **–∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ –Ω–∞—Ö–æ–¥–∏—Ç—å –∞–Ω–æ–º–∞–ª—å–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è** –±–µ–∑ —Ä—É—á–Ω–æ–π –ø—Ä–æ–≤–µ—Ä–∫–∏.  
- –ü–æ–¥—Ö–æ–¥–∏—Ç –¥–ª—è –¥–∞–Ω–Ω—ã—Ö —Å –Ω–æ—Ä–º–∞–ª—å–Ω—ã–º –∏–ª–∏ –±–ª–∏–∑–∫–∏–º –∫ –Ω–æ—Ä–º–∞–ª—å–Ω–æ–º—É —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—é.  
- –ü–æ–∑–≤–æ–ª—è–µ—Ç –æ—Ç—Ñ–∏–ª—å—Ç—Ä–æ–≤–∞—Ç—å –æ—à–∏–±–∫–∏ –∏ —á–∏—Ç–µ—Ä—Å–∫–∏–µ –∑–Ω–∞—á–µ–Ω–∏—è.  

–ï—Å–ª–∏ –Ω—É–∂–Ω–æ –Ω–∞—Å—Ç—Ä–æ–∏—Ç—å —á—É–≤—Å—Ç–≤–∏—Ç–µ–ª—å–Ω–æ—Å—Ç—å, –º–æ–∂–Ω–æ –∏–∑–º–µ–Ω–∏—Ç—å –º–Ω–æ–∂–∏—Ç–µ–ª—å `1.5` (–Ω–∞–ø—Ä–∏–º–µ—Ä, –Ω–∞ `3` –¥–ª—è –±–æ–ª–µ–µ –º—è–≥–∫–æ–π —Ñ–∏–ª—å—Ç—Ä–∞—Ü–∏–∏).

```python
    # 1. –ü—Ä–æ–ø—É—Å–∫–∏
    df['PS'].fillna(df['PS'].median(), inplace=True)
    
    # 2. –í—ã–±—Ä–æ—Å—ã
    df = df[(df['PS'] > 50) & (df['PS'] < 200)]
    
    # 3. –ö–æ–¥–∏—Ä–æ–≤–∞–Ω–∏–µ
    df = pd.get_dummies(df, columns=['rank'])
    
    # 4. –ù–æ–≤—ã–µ –ø—Ä–∏–∑–Ω–∞–∫–∏
    df['win_rate'] = df['wins'] / (df['wins'] + df['losses'])
    
    # 5. –ë–∞–ª–∞–Ω—Å–∏—Ä–æ–≤–∫–∞
    X_res, y_res = SMOTE().fit_resample(df.drop('is_cheater', axis=1), df['is_cheater'])
    
    # 6. –ù–æ—Ä–º–∞–ª–∏–∑–∞—Ü–∏—è
    X_res[['PS', 'win_rate']] = MinMaxScaler().fit_transform(X_res[['PS', 'win_rate']])
    
    return X_res, y_res
``` 
